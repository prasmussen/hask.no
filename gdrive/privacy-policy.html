Gdrive - Privacy Policy

Gdrive's privacy policy is to not share any user data and to not track anything.

Gdrive accesses user data via the google drive api and is only visible to the user who has installed gdrive and is not shared with anyone else.

The oauth access token to the drive api is stored in the .gdrive directory in the user's home directory. The token is never shared with anyone else.

The gdrive app only communicates with google drive api and only anonymous usage data is visible in the google console to the creator of gdrive.
